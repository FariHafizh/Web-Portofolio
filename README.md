# Nama Template Portofolio

Terima kasih telah membeli template ini! Ini adalah template portofolio modern berbasis React + Vite.

## Cara Menjalankan
1. Buka terminal
2. Ketik `npm install` untuk menginstall dependencies
3. Ketik `npm run dev` untuk menjalankan di browser

## Struktur Folder
- /src/components: Lokasi komponen edit
- /src/assets: Lokasi gambar
